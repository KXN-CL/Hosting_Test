# 로컬 바이브코딩 프로젝트 소개 페이지

GitHub Pages에 배포하기 위한 정적 웹사이트입니다.
국비 포트폴리오 연습용으로 제작했으며, **로컬 바이브코딩 프로젝트**의 개요와 현재 설계 진행상황을 소개합니다.

## 파일 구성

- `index.html` : 메인 페이지 구조
- `style.css` : 전체 스타일
- `script.js` : 모바일 메뉴, 스크롤 활성 메뉴, 등장 애니메이션

## GitHub Pages 배포 방법

1. 새 GitHub 저장소를 생성합니다.
2. 이 파일들을 저장소 루트에 업로드합니다.
3. GitHub 저장소의 **Settings** 로 이동합니다.
4. **Pages** 메뉴를 선택합니다.
5. **Build and deployment** 또는 **Source** 항목에서 배포 브랜치를 선택합니다.
   - 보통 `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
6. 저장 후 잠시 기다리면 GitHub Pages 주소가 생성됩니다.

## 수정 포인트

- `index.html` 안의 텍스트를 바꿔 프로젝트 설명을 더 구체화할 수 있습니다.
- `style.css` 에서 색상, 카드 모양, 간격을 조정할 수 있습니다.
- 진행률 수치는 `index.html` 의 `progress-fill` 너비 값으로 바꿀 수 있습니다.

## 목적

이 페이지는 단순한 개인 소개용이 아니라,
현재 설계 중인 로컬 AI 개발 자동화 프로젝트를 포트폴리오 형식으로 정리하기 위한 목적을 가집니다.
